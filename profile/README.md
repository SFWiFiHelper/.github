# 企鹅路由助手&星火路由助手
**企鹅路由助手** 是一个专为 "随身WiFi" 社区开发的实用工具箱, 整合了多种硬件调试和信息查看功能, 适用于多种国产随身WiFi设备 (如中兴微、紫光展锐、ASR系列等)。

**星火路由助手** 作为企鹅路由助手的进阶版本, 在原有基础上进行全面升级:

- 提供对用户更友好, 更易上手的现代化 GUI 页面
- 增加高通 410 , 高通 210 和其它路由器/随身WiFi等更多支援
- 后续将开发更多实用工具/AI功能/调试选项, 满足进阶需求
