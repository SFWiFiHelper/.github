# 企鹅路由助手&星火路由助手
**企鹅路由助手** 是一個專為 "隨身WiFi" 社區開發的實用工具箱, 綜合了多種硬體調試和資訊查看功能, 適用於多種國產隨身WiFi設備 (如中興微、紫光展銳、ASR系列等)。

**星火路由助手** 作為企鹅路由助手的進階版本, 在原有基礎上進行全面升級:

- 提供對用戶更友好, 更易上手的現代化GUI頁面
- 新增 高通410, 高通210 和其它路由器/隨身WiFi等更多支援
- 後續將開發更多實用工具/AI功能/調試選項, 滿足進階需求
