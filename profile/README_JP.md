# ペンギンルーティングアシスタント&スターファイアルーティングアシスタント
**ペンギンルーティングアシスタント** は、「モバイルWiFi」コミュニティ向けに開発されたユーティリティツールボックスで、中興マイクロ、紫光展鋭、ASRシリーズなど、さまざまな国産のモバイルWiFiデバイスに適したさまざまなハードウェアデバッグと情報表示機能を統合しています。

**スターファイアルーティングアシスタント** ペンギンルーティングアシスタントのステップアップバージョンとして、従来のベースで完全にアップグレードされました：

- ユーザーにとってよりフレンドリーで使いやすい近代的なGUIページを提供
- クアルコム410、クアルコム210、その他のルータ/モバイルWiFiなどの追加支援
- 次のステップ要件を満たすために、より多くのユーティリティ/AI機能/デバッグオプションを開発します。
